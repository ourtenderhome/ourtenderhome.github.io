# 소개

본 문서는 '호카조노 타케루'의 만화, [카구라바치]를 기반으로 하는 **'드림' 형태의 2차 창작 활동**에 대한 내용을 다룹니다.    
모든 내용은 주인공이자 드림캐인 '**로쿠히라 치히로**'와 드림주 '**로쿠히라 치사토**'를 중심으로 서술하고 있습니다.

자료용으로 본 문서를 전달받으신 경우, [세계관 및 설정](/docs/category/세계관-및-설정), [로쿠히라 치히로](./main-character/chihiro.md), [로쿠히라 치사토](./main-character/chisato.md) 문서 위주로 참고바랍니다.

:::info
#### 드림

원작이 되는 한 작품과 자작 캐릭터를 크로스오버한 IF 서사를 창작하는 행위

#### 드림캐

원작에 등장하는 인물 중 드림 활동의 중심 대상이 되는 캐릭터 

#### 드림주

원작 기반의 자작 캐릭터
:::

:::warning

문서의 모든 내용은 작성자가 원작을 보며 정리한 것으로 개인적인 해석이 들어가거나 틀린 내용이 있을 수 있습니다.

:::
